What is Programming?

Programming is the ability to write computer programs such as but not limited to websites, applications, games, and operating systems through the use of a programming language.

Why Should You Get Into Programming?

You should get into programming because not only is it a useful skill, its also fun and it can also unlock many job opportunities

What is a Programming Language?

A programming language is a form of syntax that defines how a computer should perform and what it should do.

Examples of Programming Languages:

  • C - A general purpose language commonly used to make Operating Systems but can be used for other things
  • Java - A language meant to be cross-platform and able to be run on almost any device
  • Javascript - A port of Java to the internet but is very different to Java
  • Python - A general purpose language used for servers and Artificial Intelligence.
  • and many more

Javascript Example:

console.log("Hello World!");

This is a simple command in Javascript, a programming language generally used in the creation of websites. What it does is display a message to the "console", a place where you can get information from a website. In this case it displays the message 'Hello World!' to the console.

Explaination of the Example:

So lets break it down. In console.log("Hello World!"), the 'console' part refers to, well, the console. The '.log' part refers to a function of the console that will display whatever is passed to the function onto the console. the ' "Hello World!" ' part is the string of text passed to the function. So to put it all together, you are logging "Hello World" to the console.
